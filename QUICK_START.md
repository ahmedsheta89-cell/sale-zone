# 🚀 البداية السريعة - Sale Zone Store

---

## 📌 الخطوات الأساسية

### 1️⃣ التسجيل في Vercel (دقيقة واحدة)
```
🔗 https://vercel.com/signup
👤 سجّل بـ GitHub أو Email
```

### 2️⃣ إنشاء مشروع Firebase (3 دقائق)
```
🔗 https://console.firebase.google.com
➕ Add project → "Sale Zone Store"
🌐 Add Web App → نسخ firebaseConfig
```

### 3️⃣ تحديث firebase-config.js (دقيقة واحدة)
```javascript
const firebaseConfig = {
  apiKey: "ضع_هنا",
  authDomain: "ضع_هنا",
  projectId: "ضع_هنا",
  storageBucket: "ضع_هنا",
  messagingSenderId: "ضع_هنا",
  appId: "ضع_هنا"
};
```

### 4️⃣ تفعيل Firestore (دقيقة واحدة)
```
Build → Firestore Database → Create database
Production mode → اختر Location → Enable
```

### 5️⃣ ضبط القواعد (دقيقة واحدة)
```javascript
// انسخ القواعد من FIREBASE_INTEGRATION_GUIDE.md
// Rules → Edit → Paste → Publish
```

### 6️⃣ رفع على Vercel (3 دقائق)
```
Vercel → Add New → Project
Deploy without Git → اسحب الملفات
Project Name: sale-zone-store
Deploy → انتظر → ✅
```
