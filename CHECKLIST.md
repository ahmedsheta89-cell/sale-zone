```

```

## ✅ بعد الرفع مباشرة

### الموقع الأساسي
```
□ الموقع يفتح: https://sale-zone-store.vercel.app
□ HTTPS مفعّل (🔒 في الرابط)
□ الشعار يظهر
□ الألوان صحيحة
□ النصوص العربية تظهر بشكل صحيح
□ لا توجد أخطاء 404
```

### المتجر
```
□ صفحة المتجر تفتح
□ المنتجات تظهر (حتى لو فارغة)
□ البانرات تعمل
□ السلة تعمل
□ البحث يعمل
□ الفلترة تعمل
□ القوائم تفتح بشكل صحيح
```

### لوحة التحكم
```
□ تفتح على: /ادمن_2.HTML
□ شاشة تسجيل الدخول تظهر
□ كلمة المرور الافتراضية تعمل
□ Dashboard يفتح بعد الدخول
□ القوائم تعمل
□ الإحصائيات تظهر (حتى لو صفر)
```

### Firebase
```
□ Firebase متصل (لا أخطاء في Console)
□ البيانات تُقرأ من Firestore
□ البيانات تُكتب إلى Firestore
```

### PWA
```
□ Service Worker مُسجّل
   - افتح DevTools (F12)
   - Application → Service Workers
   - يجب أن يكون: "activated and running"
□ Manifest يُحمّل بنجاح
   - Application → Manifest
   - يجب أن تظهر البيانات
□ رسالة "Add to Home Screen" تظهر (على الموبايل)
□ يمكن التثبيت كتطبيق
```

### الأمان
```
□ HTTPS مفعّل
□ غيّرت كلمة مرور الأدمن فوراً
   - لوحة التحكم → النظام → تغيير كلمة المرور
□ كلمة مرور جديدة قوية
□ كلمة المرور محفوظة في مكان آمن
```

## 🆘 إذا واجهت مشاكل

### مشكلة: الموقع لا يفتح
```
✓ تحقق من الرابط (انسخه بدون مسافات)
✓ امسح كاش المتصفح (Ctrl+Shift+Del)
✓ جرب متصفح آخر
✓ جرب جهاز آخر
✓ انتظر 5 دقائق (قد يكون النشر لم يكتمل)
```

### مشكلة: Firebase لا يعمل
```
✓ افتح Console (F12)
✓ ابحث عن أخطاء Firebase
✓ تحقق من firebase-config.js
✓ تحقق من قواعد Firestore
✓ تأكد من تفعيل Firestore
```

### مشكلة: Service Worker فشل
```
✓ تأكد من HTTPS
✓ امسح الكاش
✓ Application → Service Workers → Unregister
✓ أعد تحميل الصفحة
✓ تحقق من sw.js (لا أخطاء)
```

### مشكلة: PWA لا يُثبّت
```
✓ تأكد من HTTPS
✓ تحقق من manifest.json
✓ تحقق من Service Worker
✓ تحقق من الأيقونات (192 و 512)
✓ امسح الكاش وحاول مرة أخرى
```

### مشكلة: لوحة التحكم لا تعمل
```
✓ تحقق من كلمة المرور
✓ افتح Console → ابحث عن أخطاء
✓ تحقق من localStorage (DevTools)
✓ امسح localStorage وأعد المحاولة
```
