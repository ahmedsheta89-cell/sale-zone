## 📋 نظرة عامة

**Sale Zone Store** هو متجر إلكتروني فاخر متخصص في بيع منتجات العناية بالشعر والبشرة والطفل. المتجر مصمم بتقنيات حديثة ويدعم Progressive Web App (PWA).

---

## ✨ المميزات الرئيسية

### 🎨 **التصميم**
- تصميم فاخر بألوان ذهبية وداكنة
- واجهة مستخدم سلسة وسهلة الاستخدام
- دعم كامل للغة العربية (RTL)
- تصميم متجاوب (Responsive) على جميع الأجهزة

### 🛒 **المتجر**
- نظام سلة تسوق متطور
- قائمة المفضلات
- بحث ذكي عن المنتجات
- فلترة حسب الفئات
- تقييمات المنتجات
- صور متعددة للمنتجات

### 💳 **الدفع والطلبات**
- نظام كوبونات الخصم
- طرق دفع متعددة
- تتبع الطلبات مباشرة
- سجل الطلبات

### 💎 **نظام الولاء**
- نقاط المكافآت (5% من كل طلب)
- خصومات حصرية للعملاء المميزين
- مستويات VIP

### 📱 **PWA (Progressive Web App)**
- التثبيت على الجوال كتطبيق
- العمل بدون إنترنت (Offline)
- إشعارات push
- تحديثات تلقائية
- سرعة تحميل فائقة

### 🎛️ **لوحة التحكم (Admin Panel)**
- إدارة المنتجات (إضافة، تعديل، حذف)
- إدارة الطلبات وتتبع حالتها
- إدارة العملاء
- إدارة الكوبونات
- إدارة البانرات الترويجية
- تقارير مبيعات تفصيلية
- رسوم بيانية (Charts)
- تصدير البيانات (Excel, JSON)
- استيراد البيانات
- نسخ احتياطية كاملة

## 🚀 التثبيت والاستخدام

### المتطلبات:
- خادم ويب (Apache/Nginx)
- دعم HTTPS (ضروري لـ PWA)
- متصفح حديث يدعم Service Workers

### خطوات التثبيت:

#### 1️⃣ **رفع الملفات**
```bash
# ارفع جميع الملفات إلى السيرفر
- متجر_2.HTML
- ادمن_2.HTML
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- .htaccess (اختياري)
- robots.txt (اختياري)
- offline.html (اختياري)
```

#### 2️⃣ **تفعيل HTTPS**
PWA يتطلب HTTPS - استخدم Let's Encrypt المجاني:
```bash
# مثال: Certbot
sudo certbot --apache -d yourdomain.com
```

#### 3️⃣ **تعديل الإعدادات**
افتح `manifest.json` وعدّل:
```json
{
    "start_url": "/متجر_2.HTML",
    "scope": "/"
}
```

#### 4️⃣ **الوصول للمتجر**
```
https://yourdomain.com/متجر_2.HTML
```

#### 5️⃣ **الوصول للوحة التحكم**
```
https://yourdomain.com/ادمن_2.HTML
```

**🔐 بيانات الدخول الافتراضية:**
- كلمة المرور: `SaleZone@2024!Admin`
- ⚠️ **مهم جداً:** غيّر كلمة المرور فوراً بعد أول تسجيل دخول!

## 🔒 الأمان

### ⚠️ **مهم جداً:**

1. **غيّر كلمة مرور الأدمن فوراً**
   ```
   لوحة التحكم → النظام → تغيير كلمة المرور
   ```

2. **استخدم HTTPS دائماً**
   - PWA لا يعمل بدون HTTPS
   - حماية بيانات العملاء

3. **نسخ احتياطية دورية**
   ```
   لوحة التحكم → النظام → تصدير البيانات
   احفظ النسخة كل أسبوع على الأقل
   ```

4. **حماية ملف الأدمن**
   في `.htaccess`:
   ```apache
   <Files "ادمن_2.HTML">
       AuthType Basic
       AuthName "Admin Area"
       AuthUserFile /path/to/.htpasswd
       Require valid-user
   </Files>
   ```

## 🐛 حل المشاكل الشائعة

### 1. **Service Worker لا يعمل**
**الحل:**
- تأكد من استخدام HTTPS
- افتح Console في المتصفح
- تحقق من الأخطاء

### 2. **الأيقونات لا تظهر**
**الحل:**
- تأكد من رفع icon-192.png و icon-512.png
- تحقق من المسارات في manifest.json

### 3. **البيانات ضاعت**
**الحل:**
- استرجع من النسخة الاحتياطية:
  ```
  لوحة التحكم → النظام → استيراد البيانات
  ```

### 4. **تطبيق PWA لا يُثبّت**
**الحل:**
1. تأكد من HTTPS
2. تحقق من manifest.json
3. تحقق من Service Worker
4. امسح الكاش وأعد المحاولة
